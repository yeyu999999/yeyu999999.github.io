TestModules使用XML调用TestCase的方法（示例）。

CAPL Test Function

Call CAPL Test Function的Help解释：

1，XML test module 中的XML文件可以调用CAPL中的Test Function，且只能被XML调用。
2，调用capl中的testfunction用capltestfunction标签，且该标签无法独立使用，需被 或 或 标签包含使用。
3，以最简单的例子 <capltestfunction name="test_01" title="call test function : test_01"></capltestfunction>; name的值必须和CAPL中的testfunction 一致。title 的值可以任意填写
4，testfunction 的函数是没有返回值的。
5，capltestfunction是可以传参调用的，支持的参数有：float|int|string|signal|envvar|sysvar
